This directory contains programs and examples verified with Verus.
We report statistics about these files in the "Implementation" section of the paper.
Each file verifies with the Verus
(as of 2023-01-07, commit a58d0f36b889bb65911f7aad1e0d72ad0e48a9e1);
if a file requires special Verus flags, those are listed at the top of the file.

Verus a58d0f36b889bb65911f7aad1e0d72ad0e48a9e1 is currently available at
https://github.com/verus-lang/verus/tree/a58d0f36b889bb65911f7aad1e0d72ad0e48a9e1

Please refer to the paper artifact for a version of Verus and the examples that we
have verified to be functional: https://doi.org/10.5281/zenodo.7511038
