This directory contains full definitions and proofs for the paper's formalization.
The text files contain Unicode characters in utf8 format;
these should be readable by standard editors and web browsers.

The files are:
- definitions_utf8.txt : the complete definitions, and statements of the theorems
- definitions2_utf8.txt : typing rules rewritten using a more compact notation
- proofs_utf8.txt : proofs of preservation and progress
- proofs_termination_utf8.txt : proofs of termination
